# Internship Project - Meltronic Systemtech Pvt. Ltd.

This repository contains the assignments and projects completed during my internship at **Meltronic Systemtech Pvt. Ltd., Bangalore**. 
The internship focused on core electronics and embedded systems, involving both simulation and hardware-level design using tools like Proteus, Arduino, and VHDL.


## 🏁 Goals of the Internship
- Understand and apply embedded system fundamentals.
- Gain hands-on experience with circuit design and simulation.
- Strengthen skills in VHDL and microcontroller interfacing.
- Learn professional documentation and project workflow.

## 📌 Feedback and Next Steps
We actively sought feedback from our mentors and improved our understanding through practical experimentation. We look forward to receiving new assignments to continue learning and contributing.
